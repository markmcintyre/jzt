JZTScript Directions

Directions in JZT are evaluated by expression with the following syntax: [modifier][...] direction

Modifiers

Modifiers alter a specified direction. Here are the modifiers available in JZTScript

CW
Clockwise: Modifies a value to be clockwise from a given direction. For example, CW of North is East.
CCW
Counter-clockwise: Modifies a value to be counter-clockwise from a given direction. For example, CCW of North is West.
OPP
Opposite: Modifies a value to be the opposite of a given direction. For example, OPP of North of North is South.
RNDP
Random Perpendicular: Modifies a value to be, at random, a value perpendicular to a given direction. For example, RNDP of North may be either East or West, at random.

Directions

There are two types of directions, concrete directions and calculated directions. Concrete directions are fairly self-explanatory:

NORTH
EAST
SOUTH
WEST

Note that you can also specify these as N, E, S, and W respectively as a short-hand form.

Calculated directions require a bit of explanation:

SEEK
A direction toward the player from an object's current position.
FLOW
The same direction that an object last moved during its last command. If an object has never moved before, this value results in no direction (no movement).
RAND
A random direction, either North, South, East, or West.
RANDF
A random free direction. This is a direction relative to an object in which it is free to move. Note that objects do not consider themselves to be free to move even when an obstacle in the way is pushable.
RANDB
A random blocked direction. This is a direction relative to an object in which it is not free to move. Once again, note that objects consider themselves not to be free to move in a direction even if an obstacle is pushable.
RNDNS
A random direction, either North or South.
RNDEW
A random direction, either East or West.
RNDNE
A random direction, either North or East.

Examples

  • OPP SEEK: A direction away from the player's position.
  • RNDP FLOW: A random direction perpendicular to the direction an object last moved. For example, if an object was previously moving East, this will be interpreted as either North or South.
  • CCW RNDNE: Either North or West.
