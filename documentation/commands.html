JZT Script Commands

Introduction

Conventions

In this document, command syntax will be presented using the following conventions:

Element Description Example
Command A command is an instruction to be executed. They are displayed in bold #END
Value Values are either an expression or constant value provided to a command. They are displayed in italics #GIVE number counter
Optional values Optional values do not have to be specified by the programmer in order for a command to be successfully interpreted. These are placed in [square brackets]. #GO direction [times]
Repeatable values Some optional values may be repeated. These are indicated using an ellipsis inside square brackets like so: [...]. This indicates that the previous element may be repeated, optionally. #GO [modifier][...] direction [times]

MOVE

#MOVE direction [times]

Moves the current object in a given direction a specified number of times.
